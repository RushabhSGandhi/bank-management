# Bank Management System
This is Bank Management Web Application similar to the application used by employees in the bank.



# Technologies Used
* Frontend :- HTML, CSS, Javascript, Bootstrap 4,EJS
* Backend :- Node.js, Express.js
* Database :- MySQL
* Storage :- Firebase
* Hosting :- Heroku
